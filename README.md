# Academic Homepage

Personal academic homepage built as a modern static React site with Vite.

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/`.

## Content

Editable homepage content lives in `src/content/`. Images live in `public/images/`.

## Build

```bash
npm run build
npm run preview
```

The production output is generated in `dist/`.
