# Academic Homepage

Personal academic homepage built as a modern static React site with Vite.

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/`.

## Content

Editable homepage content lives in `src/content/`. Images live in `public/images/`, including profile photos, publication figures, favicon assets, and social preview images. For favicon generation, [redketchup favicon-generator](https://redketchup.io/favicon-generator) works well; copy the generated files into `public/images/`.

## Build

```bash
npm run build
npm run preview
```

The production output is generated in `dist/`.
